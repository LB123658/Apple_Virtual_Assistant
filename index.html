How to save as an app:

1. Download the file and open it (it will automatically open in Script Editor)
2. To run the app out of Script Editor click the play button in the upper right area of the app.
3. Click on "File" in the top menu and click on "Save"
4. Title the file as "Bob" and select "application" to save it as. *It will probably say that it is an AppleScript file.
5. Now it is saved as an app and you can move it to your Applications Folder. Simply click on it to use Bob, the virtual assistant.
